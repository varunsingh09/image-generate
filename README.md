## :white_check_mark: Requirements ##

```bash
# Clone this project
$ git clone https://github.com/varunsingh09/image-generate

# Access
$ cd assignment

# Install dependencies
$ yarn

# Run the project
$ npm run dev

# The server will initialize in the <http://localhost:3000>
```
